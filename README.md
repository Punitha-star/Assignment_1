#  Earthquake Data Analysis Application

## Project Overview
This project analyzes earthquake data using Python, Streamlit, and SQLAlchemy.
It provides trend analysis, tsunami comparison.

## Technologies Used
- Python
- Streamlit
- SQLAlchemy
- Pandas
- MySQL 

##  Features
- Real-time data fetching (BASE_URL = "https://earthquake.usgs.gov/fdsnws/event/1/query")
-  Top affected countries identification
-  Database connectivity using SQLAlchemy 
- Tsunami vs Non-tsunami magnitude comparison
- Depth analysis of earthquakes
- Interactive dashboard using Streamlit

##  How to Run the Project

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   streamlit run streamlit_eq.py
